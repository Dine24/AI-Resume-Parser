# AI Resume Agent

An AI-powered Resume Analysis and ATS Optimization tool built using Python, Streamlit, and Groq LLMs.

## Features
- Resume PDF Parsing
- ATS Score Calculation
- AI-generated Professional Summary
- Missing Keyword Detection
- Job Description Matching

## Tech Stack
- Python
- Streamlit
- Groq API
- Scikit-learn
- PyPDF2

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
